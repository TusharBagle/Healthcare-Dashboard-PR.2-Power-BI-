## 📌 Project Overview

The **Healthcare Dashboard** is an interactive Power BI report designed to analyze hospital operations, patient admissions, billing performance, medical conditions, and flight-style KPI visualization techniques for healthcare data.

The dashboard enables users to quickly monitor hospital performance through dynamic KPIs, interactive charts, slicers, drill-through pages, and detailed patient information.

---

# 📊 Dashboard Features

### Executive Dashboard

- 👨‍⚕️ Total Patients
- 💰 Total Billing Amount
- 📈 Average Billing Amount
- 🛏️ Average Length of Stay
- 🚹 Male vs Female Distribution
- 🏥 Department Analysis
- 📅 Monthly Admission Trends
- 📍 Admission Type Analysis
- ❤️ Medical Condition Analysis
- 🛡️ Insurance Provider Analysis
- 🚨 Patient Risk Indicators
- 🎯 Interactive KPI Cards

---

### Patient Detail Page

Provides detailed patient-level information including:

- Patient Name
- Age
- Gender
- Medical Condition
- Department
- Admission Type
- Length of Stay
- Billing Amount
- Insurance Provider
- Test Results
- Risk Flag

---

# 📈 Dashboard Insights

The report helps answer questions such as:

- Which department has the highest number of patients?
- Which medical condition is most common?
- What is the total hospital billing?
- Which insurance provider covers the most patients?
- What is the monthly admission trend?
- How are patients distributed by gender?
- Which admission type occurs most frequently?
